# Sales and Customer Analytics

## Overview
This project analyzes retail sales data to uncover revenue trends, top-performing products, and customer purchasing behavior. The project applies data analytics techniques learned from Cisco NetAcad's Data Analytics Essentials course.

## Dataset
Public retail sales dataset (CSV format)

## Objectives
- Analyze monthly revenue trends
- Identify best-selling product categories
- Segment customers based on spending behavior
- Generate business insights for decision-making

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Findings
### Key KPIs (calculated):
- Total sales: 2297200.86
- Mean order value: 229.858
- Total profit: 286397.02

**High Value Customers**:
- 204 customers
- Generated ~1,004,194 in revenue
- Smallest group, but highest revenue contribution per customer

**Medium Value Customers**:
- 296 customers
- Generated ~975,346 in revenue
- Strong growth potential with marketing and retention strategies

**Low Value Customers**:
- 293 customers
- Generated ~317,661 in revenue
- Largest risk group for churn and low profitability
